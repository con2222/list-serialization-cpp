# Сериализация и десериализация двусвязного списка

Решение тестового задания на позицию C++ разработчика.
Программа реализует алгоритм сериализации и десериализации двусвязного списка в бинарный файл, где каждый узел содержит случайный указатель (`rand`) на любой другой узел списка.

## Сборка и запуск (Linux / macOS / WSL)

```bash
g++ main.cpp -o solution
./solution
```

## Формат данных
* **Входной файл (`inlet.in`)**: Текстовый формат. Каждая строка описывает один узел как `<data>;<rand_index>`.
* **Выходной файл (`outlet.out`)**: Бинарный формат. Структура записи каждого узла:
    1. `size_t` (8 байт) — длина строки `data`.
    2. `char*` (N байт) — содержимое строки.
    3. `int` (4 байта) — индекс узла, на который ссылается `rand` (-1, если указатель пустой).

## Ключевые функции
* **`serialize_list`**: Проходит по списку и сохраняет состояние каждого узла в бинарный файл. Для перевода указателей в индексы используется `std::unordered_map`, что обеспечивает общую скорость работы **O(N)**.
* **`deserialize_list`**: Считывает бинарный файл, восстанавливает структуру двусвязного списка и реконструирует связи `rand` по сохраненным индексам за два прохода (создание узлов + связывание).
* **`clear_list`**: Гарантирует полное освобождение памяти, выделенной под узлы списка.

## Отладка
Для проверки корректности восстановления списка в коде предусмотрена функция:
* **`print_list`**: Выводит содержимое списка в консоль в формате `Data -> RandomData`. Это позволяет визуально убедиться, что после десериализации все случайные связи (указатели `rand`) указывают на те же данные, что и до сохранения.
